## Design Guidelines

### اصول طراحی:
1. **تطابق Divar**: 
   - Card-based layout
   - Minimal decorations
   - Focus on content
   - Clear typography hierarchy

2. **RTL/Farsi First**:
   - All text right-aligned
   - Icons on correct sides
   - Proper spacing

3. **Performance**:
   - Image lazy loading
   - Smooth animations (200-300ms)
   - No unnecessary re-renders

4. **Accessibility**:
   - Sufficient color contrast
   - Tap targets min 44x44px
   - Proper semantic HTML/widgets

5. **Mobile First**:
   - Portrait orientation primary
   - Touch-friendly buttons
   - Responsive spacing
