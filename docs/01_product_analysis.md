# VAYA - محصول تحلیل و MVP

## 1. هویت برند
- **نام**: VAYA (وایا) - سبزوار سروسز دایرکتوری
- **ماتو**: "سرعت، اعتماد، ارتباط"
- **رنگ اصلی**: آبی حرفه‌ای (#007AFF) + سفید
- **رنگ ثانویه**: نارنجی‌گرم (#FF6B35) برای تاکید
- **منطق طراحی**: Divar-inspired ساده، تمیز و قابل اعتماد

---

## 2. تجزیه نیازمندی‌ها

### الف) مخاطبان
| نقش | مجوزها | محدودیت |
|------|---------|----------|
| **کاربر عادی** | جستجو، مرور، علاقه‌مندی، اشتراک | فقط مشاهده |
| **مدیر** | CRUD کامل، مدیریت فیلترها | دسترسی پنل وب |
| **سیستم** | پردازش تصاویر، اعلان، جستجو | بدون مداخله دستی |

### ب) ویژگی‌های اصلی
1. **جستجوی قدرتمند**: متن کامل، دسته‌بندی، منطقه
2. **فیلترینگ**: شهر، منطقه، دسته‌بندی، امتیاز، نوع آگهی
3. **جزئیات کسب‌وکار**: تمام اطلاعات تماس، نقشه، گالری
4. **علاقه‌مندی**: ذخیره برای دسترسی سریع
5. **اشتراک‌گذاری**: اشتراک‌گذاری سریع
6. **نظرات و امتیاز**: کاربران می‌توانند نظر دهند
7. **اعلان‌ها**: هنگام ایجاد آگهی جدید
8. **پنل مدیریت**: ساخت و مدیریت آگهی‌ها

## 3. MVP (نسخه اول قابل ارسال)

### Phase 1 - MVP Core
#### Frontend (Flutter)
- [ ] صفحه اصلی با جستجو
- [ ] صفحه لیست (infinite scroll)
- [ ] صفحه جزئیات کسب‌وکار
- [ ] صفحه علاقه‌مندی‌ها
- [ ] صفحه دسته‌بندی‌ها
- [ ] صفحه مناطق
- [ ] UI کاملاً فارسی و RTL
- [ ] Loading/Empty/Error states

#### Backend (Node.js + Express)
- [ ] API مدل‌های اصلی
- [ ] جستجوی Full-Text
- [ ] Authentication (مدیر)
- [ ] Database Schema
- [ ] File Upload (تصاویر)

#### Admin Panel (React)
- [ ] CRUD آگهی‌ها
- [ ] Upload تصاویر
- [ ] مدیریت دسته‌بندی‌ها
- [ ] مدیریت مناطق
- [ ] Dashboard ساده

#### Database (PostgreSQL)
- [ ] Schema طراحی‌شده
- [ ] Indexes برای جستجو
- [ ] Relations تعریف‌شده

## 5. داده‌های موجود و پیش‌تعریف

### دسته‌بندی‌های خدمات فنی (Seed Data)
1. تعمیر و نگهداری (رایانه، موبایل)
2. نصب و راه‌اندازی
3. آموزش و مشاوره
4. توسعه نرم‌افزار
5. خدمات شبکه و سرور

### مناطق سبزوار (Seed Data)
- مرکز شهر
- بخش شمالی
- بخش جنوبی
- بخش شرقی
- بخش غربی
- حومه

## 6. معیارهای موفقیت MVP

- ✓ اپ بدون خرابی باز شود
- ✓ جستجو سریع‌تر از 500ms
- ✓ لیست 20 آگهی در کمتر از 1 ثانیه لود شود
- ✓ پنل مدیریت 5 دقیقه‌ای یک آگهی ایجاد کند
- ✓ UI 100% RTL و فارسی
- ✓ APK < 50MB
- ✓ هیچ ارور در Logcat
